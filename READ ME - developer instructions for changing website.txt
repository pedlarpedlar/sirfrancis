To change payfast details from testing mode to live:
	Update the following variables to those found on client's Developer Settings at my.payfast.io:
		notify.php
		1.	define('SANDBOX_MODE', true); 		-> define('SANDBOX_MODE', false);
		2.	$pfPassphrase = 'jt7NOE43FZPn'; 	-> $pfPassphrase = 'clientpasswword'; // make one for client. this is set in Developer Settings on my.payfast.io

		payNowForm.php AND profile.php
		3.	$testingMode = true;				-> $testingMode = false;
		4.	$passphrase = 'jt7NOE43FZPn'; 		-> $passphrase = 'clientpasswword'; // make one for client. this is set in Developer Settings on my.payfast.io
		5.	'merchant_id' => '10000100', 		-> 'merchant_id' = 'clientmerchantid', //this is found in Developer Settings on my.payfast.io
		6.  'merchant_key' => '46f0cd694581a', 	-> 'merchant_key' = 'clientmerchantkey', //this is found in Developer Settings on my.payfast.io

Showing products on Google (free) shopping tab:
	Client must have a Merchant Account center setup (https://myaccount.google.com/?utm_source=sign_in_no_continue&pli=1&nlr=1)
	Search Merchant Center for "feed" or navigate to "Data Sources"
	Add product source (use URL, because we are generating the products daily.)
	Url should be something like website.co.za/uploads/google_products/google.txt or point to wherever the file is stored. This file is generated by candybird.co.za/generate_google_shopping_items
